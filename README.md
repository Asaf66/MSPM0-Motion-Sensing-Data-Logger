# MSPM0 Motion-Sensing & Data-Logger Platform

### Project Overview
A compact, high-performance embedded system designed for precise motion tracking and data acquisition, built around the Texas Instruments MSPM0 series MCU.

### Technical Highlights
* Full PCB Design: Handled the entire process from schematic to layout, focusing on stable power delivery (LDO) and signal routing.
* Embedded Hardware: Integrated a TI MSPM0 series MCU, including USB-C connectivity and a USB-to-UART bridge for easy data logging.
* Sensor Integration: Implemented I2C communication for motion tracking (accelerometer) and managed various digital/analog interfaces like ADC and SPI.
